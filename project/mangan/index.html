ManGan

Mar 2, 2019
Star
View source View paper

This project aims to color black and white mangap ages, based on the corresponding anime adaptation, using Generative Adversarial Nets.

For the moment, two types of GANs have been tested: pix2pix and cycle-gan.

For more information about our method, see the Project Report.
